# Restaurante Cozinha Goiana

## Sobre o Projeto

O Restaurante Cozinha Goiana é um projeto acadêmico desenvolvido para a disciplina de Front-End da Faculdade Católica. O objetivo do projeto é criar um site moderno, responsivo e intuitivo para apresentar o restaurante, seu cardápio, informações de contato e proporcionar uma melhor experiência aos clientes.

## Funcionalidades

* Página inicial com apresentação do restaurante
* Cardápio online
* Informações sobre a história do restaurante
* Contato e localização
* Navegação responsiva para diferentes dispositivos

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo

Desenvolver uma aplicação web utilizando os conceitos de desenvolvimento Front-End, aplicando boas práticas de estruturação, estilização e experiência do usuário.
